📚 BookManager

BookManager는 React, TypeScript, Vite를 사용하여 구축된 간단한 웹 애플리케이션입니다. 이 프로젝트는 빠른 개발 환경 설정을 위해 Vite를 사용하며, 타입 안전성을 위해 TypeScript를 채택하였습니다.

⸻

🚀 시작하기

필수 조건
	•	Node.js (권장 버전: 18 이상)
	•	npm 또는 yarn

설치 및 실행
# 저장소 클론
git clone https://github.com/crimsonpinus/bookmanager.git

# 디렉토리 이동
cd bookmanager

# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
